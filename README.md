# ❌⭕ Tic Tac Toe Game (Java)

A simple and interactive **Tic Tac Toe game built using Java**.
This project allows two players to play the classic Tic Tac Toe game with a clean interface and smooth gameplay logic.

---

## 🎮 About the Game

Tic Tac Toe is a classic two-player strategy game where players take turns marking **X** and **O** on a 3×3 grid.
The player who succeeds in placing **three of their marks in a horizontal, vertical, or diagonal row wins the game**.

---

## 🚀 Features

* 🎮 Two-player gameplay
* ❌⭕ Automatic turn switching (X and O)
* 🏆 Winner detection system
* 🔄 Game reset option
* 🎨 Simple and user-friendly interface
* ⚡ Fast and lightweight Java application

---

## 🛠️ Technologies Used

* **Java**
* **Java Swing / AWT (GUI)**
* **Object-Oriented Programming (OOP)**
